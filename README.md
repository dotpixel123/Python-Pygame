# Snake Game

A classic Snake game built with Python and Pygame.

## About This Project

This game was built from scratch, without any AI assistance. The game mechanic and features was designed and implemented through my own learning and problem-solving.

## Features

- Classic snake gameplay with smooth movement
- Collision detection with walls and self
- Score tracking with persistent high score system
- Custom assets and visuals
- Responsive controls

## How to Run

```bash
uv run snake.py
```

## Requirements

- Python 3.x
- Pygame
- Pygame image assets in the `Assets/` folder

## Controls

- **Arrow Keys**: Move the snake
- **ESC**: Exit the game
